Chain Reaction Cycle - Revamp

Redesign - 2019

Context

Wiggle recently acquired Chain Reaction Cycle (CRC) and created a project called BMX to revamp the entire website to bring it up to date.

Business Opportunities

  • Improve CRC design/brand
  • Create a style guide for the website to make the site more consistent in design
  • Identify current user issues
  • Support the responsive programme
  • Improve accessibility
  • Increase organic traffic

Current Experience

The current website is messy and cluttered as well as outdated compared to the current Wiggle website. Some features on the site are also hard to find and difficult to use.

Assumptions

The biggest restriction on this project was the time limit. Each epic was given about one week to research, design, prototype and test.

The Design Phase

As we were very restricted on time I was given the epics I had knowledge of due to researching and designing them for Wiggle’s website. As these had been done fairly recently it meant I could apply this knowledge and skip straight to designing and then prototyping.

Conclusion

Having the research knowledge from Wiggle’s site made the designing and prototyping phase a lot quicker. So when testing with users very few alterations needed to be made. I’ve also moved on to a new role so wishing all the best to the UX/UI team in Wiggle with getting this massive project finished.
